# 00-meta

元能力层，放那些贯穿整个设计工作流的能力，而不是只服务某个单一阶段的技能。

## 当前已实现

- `thinking-coach/`
- `systems-design-advisor/`

## 计划中的方向

- `systems-thinking`
- `design-critique`

## 说明

当前的 `thinking-coach` 已经补上了元能力层里“思考教练”这一条线；`systems-design-advisor` 暂时承担了计划里 `systems-thinking` 这条线的职责。等后续方法论和资源更完整时，可以考虑升级或重命名，但现阶段先保持发布名稳定。对应的安装包统一放在仓库根目录的 `packages/` 下。
